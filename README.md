# 🩸 HemoVault: Blood Bank Management System

**HemoVault** is a premium, full-stack management solution designed to streamline the operations of a blood bank. From tracking donors and managing hospital requests to real-time inventory monitoring, HemoVault ensures that "preserving the flow of life" is handled with precision and security.

## ✨ Features

- **🔐 Secure Access**: Role-based authentication with encrypted sessions for authorized personnel.
- **📊 Interactive Dashboard**: Real-time metrics for current blood stock, total donors, and pending hospital requests.
- **🩸 Inventory Management**: Automated tracking of blood units across all major blood groups (A+, B+, O+/-, etc.).
- **🏥 Hospital Integration**: Manage hospital profiles and streamline blood requests with approval workflows.
- **🤝 Donor Tracking**: Complete donor database including contact info, blood type, and last donation date.
- **🛡️ Safety Rules**: Built-in logic to ensure safety intervals between donations.
- **📈 Professional Reporting**: Generate system-wide reports for inventory and request history.

---

## 🛠️ Tech Stack

- **Backend**: Node.js & Express.js
- **Database**: MySQL (with `mysql2/promise`)
- **Frontend**: EJS (Embedded JavaScript Templates)
- **Styling**: Vanilla CSS & Bootstrap 5 (Modern Dark/Cyber-Security Theme)
- **Security**: Bcrypt for password hashing & Express-session for authentication.
